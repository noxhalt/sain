# sain - Simple Aur Installer
Just a install only aur helper.

## How to use

```
./sain [aur-packages...]
```

## Why?

I heard we can download *Plaintext PKGBUILD* from AUR.  
So, I tried creating simple aur installer.

## Feature

Install only!  
You can `./sain yay-bin`.

## Warning

Using AUR is at your own risk.
